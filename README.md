
# google-search-cache

A simple application for caching site search results from google search. You can use it as an api to search your site.

## Installation

```bash
$ npm install
```

## Config

Specify TARGET_URL variable in .env file

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
